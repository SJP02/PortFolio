NOTES

NOTES, which stand for Natively Orented Textual Enrichment System is an application that can be activated by users within a single click. The application allows student users to copy link of the desired youtube videos or aloow teachers to record their audio lectures. In the students perspective, one has to paste the url of the desired youtube video which will then to converted to text and further summarized into notes and will be available to the student in the pdf format. In the teacher's perspective, there will be a record button displayed which has to be clicked befpre giving out the lectures. The audio will be converted to text, symmarized and late will be available in pdf format that can be distributed to students as notes.

Decentra-Law

Decentra-Law is a Dapp, that is, an application using blockchain technology, that decentalizes the procedure of acquiring BAR council licenses through on-chain relaying and verification of the documents and licenses using smart contracts. It mainly aim to tackle the hastle of mandatory physical presence of advocates for issuing BAR Council licenses. Digitalizing this data abstracts the unwanted data that the user usually has to provide otherwise, i.e., when done physically.

Land Registration Using Fractional NFT

It replaces the traditional land registration process by digitalizing it using blockchain technology, more specifically using fractional NFT. It makes the process more transparent, secure and corrupt-free.This approach allows land to be divided and shared among multiple stakeholders. It empowers individuals to invest in and trade portions of the land, democratizing access to the real estate market. It also helps in reducing overhead of storing and retrieving documents in remote locations that lack infrastructure.
